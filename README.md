# Employee Performance Analysis Dashboard

## Overview
This project demonstrates data visualization best practices using a ChatGPT-assisted workflow for creating compelling data visualizations from employee performance data.

## Project Details
- **Email Verification**: 21f3001699@ds.study.iitm.ac.in
- **Dataset**: 100 employees across 6 departments and 5 regions
- **Focus**: Sales Department Analysis

## Files Included
- `employee_analysis.py` - Python script for data loading, analysis, and visualization
- `employee_analysis.html` - Interactive web-based visualization dashboard
- `employee_analysis.png` - Static visualization image

## Features
The visualization includes:
1. **Department Distribution Histogram** - Shows employee count by department
2. **Performance Score Analysis** - Distribution of performance metrics
3. **Departmental Performance Comparison** - Average scores by department
4. **Regional Distribution** - Geographic breakdown of workforce

## How to Use
1. View the dashboard by opening `employee_analysis.html` in a web browser
2. The HTML file is self-contained and works offline
3. All visualizations are interactive and responsive

## Technical Stack
- Python 3.x
- Pandas - Data manipulation
- Matplotlib & Seaborn - Visualization
- Chart.js - Interactive web charts
- HTML/CSS/JavaScript - Web interface

## ChatGPT Workflow
This project demonstrates the best practices outlined in "Prompt to Plot" methodology:
1. Dataset discovery and creation
2. Story ideation based on available columns
3. Statistical analysis and visualization
4. Web-ready code generation with performance considerations
5. Professional design and styling
